# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-03-17

### Added
- Initial release
- `HumaneType` base component
- `HumaneTitle` component for title text effects
- `HumaneBody` component for body text effects
- TypeScript definitions
- Demo application

### Features
- Title text effects (character rotation, baseline shift, tracking variation, size variation)
- Body text effects (text on curved path)
- Customizable effect intensities
- Accessibility support
- React 18 and 19 compatibility
