# Humane

A modern React UI component library built with TypeScript and Vite.

## Installation

```bash
npm install humane
# or
yarn add humane
# or
pnpm add humane
```

## Usage

```jsx
import { App } from 'humane';
// or import the default export
import Humane from 'humane';

function MyComponent() {
  return (
    <div>
      <App />
      {/* or */}
      <Humane.App />
    </div>
  );
}
```

## Features

- Modern React components
- TypeScript support
- Tree-shakeable exports
- Fully typed components

## Development

To start the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

### Build

To build the library:

```bash
npm run build
# or
yarn build
# or
pnpm build
```
